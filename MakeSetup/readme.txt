The program starts automatically in the background at login and is only visible as an icon in the traybar.
Now it is watching the running application and temporary disables system's power saving as long as one of the configured applications is found like video recording. Then the icon is highlighted.

The configuration window is shown by double-clicking the icon. The running user applications are listed and may be selected for getting monitored. Please use the minimize button to finish configuration and continue monitoring. Closing the window will stop the program.
